Alfari Software


Alfari Software adalah Perusahaan Developer Aplikasi dan Game yang berbasis di Yogyakarta. Alfari Software berdiri pada bulan Oktober 2018 didirikan oleh Aliffathur Risqi Hidayat seorang mahasiswa jurusan Teknik Informatik di Universitas Teknologi Yogyakarta.

Awal mula terbentuknya Alfari Software bermula dari passion sang pendiri (Aliffathur) yang sangat mencintai Game, Dia tertantang untuk tidak hanya bermain game dari buatan orang lain tapi juga buatannya sendiri, berawal dari sebuah keinginan itu Dia mencoba mencari orang yang memiliki passion yang sama dengannya.

Alfari Software sendiri memiliki beberapa target selain membuat game mereka juga mencoba untuk membuat aplikasi mobile maupun PC yang diharapkan bisa berkembang dan dapat diterima semua kalangan masyarakat, tetapi untuk saat ini Alfari Software sedang mengembangkan game PC pertama mereka.
